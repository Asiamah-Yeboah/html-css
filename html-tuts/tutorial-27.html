What is Flexbox?

Flexbox (Flexible Box Layout) is a one-dimensional layout model designed for distributing space and aligning items in a container.

Traditional Layout

  • Uses floats, display, position
  • Complex for vertical centering
  • Clearfix hacks needed
  • Unequal height columns difficult
  • Source order dependent

Flexbox Layout

  • Single property: display: flex
  • Easy vertical centering
  • No clearfix needed
  • Equal height columns automatic
  • Reorder items visually
/* The magic starts with one line */
.container {
  display: flex;
  /* That's it! Children become flex items */
}

The Two Axes

Flexbox operates on two axes:

  • Main Axis: Primary direction (row or column)
  • Cross Axis: Perpendicular to main axis

All Flexbox properties align items along or across these axes. Understanding this is key to mastering Flexbox!

[2:30 - 5:30]

Flex Container Properties

These properties are set on the container (parent element):

/* Direction - Defines main axis */
.container {
  flex-direction: row; /* Default: left to right */
  flex-direction: row-reverse;
  flex-direction: column; /* Top to bottom */
  flex-direction: column-reverse;
}

/* Wrap - Allows items to wrap to new lines */
.container {
  flex-wrap: nowrap; /* Default: single line */
  flex-wrap: wrap; /* Multi-line */
  flex-wrap: wrap-reverse;
}

/* Justify Content - Alignment along MAIN axis */
.container {
  justify-content: flex-start; /* Default */
  justify-content: flex-end;
  justify-content: center;
  justify-content: space-between;
  justify-content: space-around;
  justify-content: space-evenly;
}

/* Align Items - Alignment along CROSS axis */
.container {
  align-items: stretch; /* Default: fill container */
  align-items: flex-start;
  align-items: flex-end;
  align-items: center; /* Vertical centering! */
  align-items: baseline; /* Align text baselines */
}

The Famous Centering Trick

With traditional CSS, vertical centering was notoriously difficult. With Flexbox, it's one line:

/* Perfect centering: horizontal AND vertical */
.centered {
  display: flex;
  justify-content: center; /* Horizontal */
  align-items: center; /* Vertical */
}

This alone makes Flexbox worth learning! No more margin: auto hacks or transform tricks!

[5:30 - 8:00]

Flex Item Properties

These properties are set on individual items (child elements):

/* Flex Grow - How item grows relative to others */
.item-1 { flex-grow: 1; } /* Takes available space */
.item-2 { flex-grow: 2; } /* Takes twice as much space */

/* Flex Shrink - How item shrinks relative to others */
.item { flex-shrink: 1; } /* Default: can shrink */
.no-shrink { flex-shrink: 0; } /* Won't shrink */

/* Flex Basis - Initial size before growing/shrinking */
.item {
  flex-basis: 100px; /* Fixed width */
  flex-basis: auto; /* Default: uses width/height */
  flex-basis: 25%; /* Percentage of container */
}

/* Flex Shorthand - grow | shrink | basis */
.item {
  flex: 1 1 auto; /* Default */
  flex: 1; /* Same as 1 1 0 */
  flex: 0 0 200px; /* Fixed size, won't grow/shrink */
}

Visual reordering!

/* Align Self - Override container's align-items */
.special-item {
  align-self: flex-start;
  /* Options: auto | flex-start | flex-end | center | baseline | stretch */
}

/* Order - Visual reordering */
.first-visually { order: -1; } /* Appears first */
.last-visually { order: 999; } /* Appears last */

The Flex Shorthand

The flex shorthand can be confusing but powerful:

  • flex: initial = 0 1 auto (won't grow, can shrink)
  • flex: auto = 1 1 auto (fully flexible)
  • flex: none = 0 0 auto (inflexible)
  • flex: 1 = 1 1 0 (equal distribution)
  • flex: 0 0 200px (fixed 200px, won't grow/shrink)

Pro Tip: Use flex: 1 for equal-width columns!

Real World Flexbox Examples

Let's build some common layouts with Flexbox:

/* 1. Modern Navigation Bar */
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 1rem;
}

/* 2. Holy Grail Layout (Header, Footer, 3 Columns) */
.holy-grail {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}
.main-content {
  display: flex;
  flex: 1;
}
.sidebar { flex: 0 0 250px; }

/* 3. Card Grid with Equal Height Cards */
.card-grid {
  display: flex;
  flex-wrap: wrap;
  gap: 20px; /* Modern spacing */
}
.card {
  flex: 1 1 300px; /* Grow, shrink, min 300px */
  display: flex;
  flex-direction: column;

Responsive Flexbox

Flexbox makes responsive design incredibly easy:

/* Responsive navigation */
.nav-menu {
  display: flex;
  flex-direction: row;
}

@media (max-width: 768px) {
  .nav-menu {
    flex-direction: column;
  }
}

With one property change, your horizontal navigation becomes vertical on mobile!

[10:00 - 11:15]

Common Flexbox Patterns

Must-Know Flexbox Patterns

1. Equal Height Columns:

.columns {
  display: flex;
}
.column {
  flex: 1; /* All equal height and width */
}

2. Sticky Footer:

body {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}
main {
  flex: 1; /* Pushes footer down */
}

3. Perfectly Centered Modal:

.modal {
  display: flex;
  justify-content: center;
  align-items: center;
  position: fixed;
  top: 0; left: 0;
  width: 100%; height: 100%;
}

Remember: These patterns solve problems that used to require complex CSS hacks!
