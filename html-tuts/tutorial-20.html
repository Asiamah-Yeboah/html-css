1. Complete HTML Document Structure

The foundation of every HTML page:

<!-- Modern HTML5 Document Structure -->
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Page Title Here</title>
  <meta name="description" content="Page description for SEO">
  <link rel="stylesheet" href="styles.css">
  <link rel="icon" href="favicon.ico">
</head>
<body>
  <!-- Your content goes here -->
  <h1>Hello, World!</h1>
</body>
</html>

    2. Semantic HTML Elements

    Structure your content meaningfully:

    4

    Document Structure

  • <header> Introductory content
  • <nav> Navigation links
  • <main> Primary content
  • <footer> Closing content
    • 5

    Content Sections

  • <article> Self-contained content
  • <section> Thematic grouping
  • <aside> Related content
  • <div> Generic container
    • 6

    Media & Figures

  • <figure> Self-contained media
  • <figcaption> Figure caption
  • <time> Date/time (machine-readable)
  • <address> Contact information

    • Semantic Rule

    Use semantic elements for their intended purpose. Don't use <div> for everything. Semantic HTML improves accessibility, SEO, and maintainability.
