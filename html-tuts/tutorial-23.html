CSS Color Systems

CSS gives you multiple ways to define colors. Each has its own use case:

/* 1. Named Colors - 140+ predefined names */
h1 { color: red; }
p { color: cornflowerblue; }

/* 2. HEX Colors - #RRGGBB or #RGB */
.primary { color: #ff5733; } /* Full hex */
.secondary { color: #f50; } /* Short hex */

/* 3. RGB - Red, Green, Blue (0-255) */
.alert { background-color: rgb(255, 87, 51); }

/* 4. RGBA - RGB with Alpha (transparency) */
.overlay { background-color: rgba(0, 0, 0, 0.5); }

/* 5. HSL - Hue, Saturation, Lightness */
.accent { color: hsl(12, 100%, 60%); }

/* 6. HSLA - HSL with Alpha */
.glass { background: hsla(0, 0%, 100%, 0.2); }

When to Use Which?

  • HEX: Most common, great for solid colors
  • RGB/RGBA: When you need transparency
  • HSL/HSLA: When adjusting brightness/saturation (easier than HEX!)
  • Named colors: Quick prototyping, but limited choices

Pro Tip: HSL is often the easiest to work with because you can adjust lightness without changing the hue!

[2:30 - 4:15]

Background Properties

CSS gives you complete control over backgrounds. Let's explore the key properties:

/* Individual background properties */
.hero {
  background-color: #f0f0f0; /* Solid color */
  background-image: url('hero.jpg'); /* Image */
  background-repeat: no-repeat; /* Repeat control */
  background-position: center center; /* Positioning */
  background-size: cover; /* Size control */
  background-attachment: fixed; /* Scroll behavior */
}

/* Shorthand background property */
.banner {
  background: #333 url('pattern.png') center/cover no-repeat;
}

Background Size Magic

The background-size property is incredibly powerful:

  • cover: Scales image to cover entire element (may crop)
  • contain: Scales image to fit inside element (no cropping)
  • 100% 100%: Stretches to fill (distorts image)
  • 200px 150px: Exact dimensions
  • 50% auto: 50% width, proportional height

Use 'cover' for hero sections and 'contain' for logos!

[4:15 - 6:00]

CSS Gradients

Gradients are one of the most beautiful features of modern CSS. No images needed!

/* 1. Linear Gradient - goes in one direction */
.gradient-1 {
  background: linear-gradient(to right, #ff7e5f, #feb47b);
}

.gradient-2 {
  background: linear-gradient(45deg, #667eea, #764ba2);
}

/* 2. Radial Gradient - radiates from center */
.gradient-3 {
  background: radial-gradient(circle, #ff7e5f, #feb47b);
}

/* 3. Conic Gradient - rotates around center */
.gradient-4 {
  background: conic-gradient(red, yellow, green, blue, red);
}

/* 4. Multiple Color Stops */
.gradient-5 {
  background: linear-gradient(to right, red 0%, orange 25%, yellow 50%, green 75%, blue 100%);
}

/* 5. Repeating Gradient */
.gradient-6 {
  background: repeating-linear-gradient(45deg, #667eea, #667eea 10px, #764ba2 10px, #764ba2 20px);
}

Gradient Design Tips

  • Use subtle gradients for buttons and cards
  • Direction matters: left-to-right feels calm, diagonal feels dynamic
  • Limit colors: 2-3 colors usually work best
  • Test accessibility: Ensure text remains readable
  • Check browser support: Use vendor prefixes for older browsers

Gradients can transform a flat design into something modern and engaging!

[6:00 - 7:00]

Multiple Backgrounds

You can layer multiple backgrounds on a single element! This is incredibly powerful for creating complex effects.

/* Multiple backgrounds separated by commas */
.hero-section {
  background:
    linear-gradient(rgba(0, 0, 0, 0.5), rgba(0, 0, 0, 0.5)),
    url('hero-image.jpg') center/cover
  ;
  color: white;
  min-height: 400px;
}

/* Three layers: color, gradient, image */
.complex-bg {
  background:
    linear-gradient(45deg, rgba(255,255,255,0.1) 25%, transparent 25%),
    linear-gradient(-45deg, rgba(255,255,255,0.1) 25%, transparent 25%),
    linear-gradient(45deg, transparent 75%, rgba(255,255,255,0.1) 75%),
    linear-gradient(-45deg, transparent 75%, rgba(255,255,255,0.1) 75%),
    #4b6cb7
  ;
  background-size: 20px 20px;
}
') center/cover;">
Overlay Effect Example

Layer Order Matters!

When using multiple backgrounds, the first layer is on top and the last layer is on the bottom.

Common patterns:

  • Gradient overlay on image (for better text contrast)
  • Pattern over solid color (for texture)
  • Multiple gradients (for complex effects)
  • SVG patterns as backgrounds
[7:00 - 7:45]

Color Accessibility

Making Colors Accessible

About 1 in 12 men and 1 in 200 women have some form of color blindness. Your color choices matter!

Good Contrast: #4b6cb7 on white
Poor Contrast: #b3c1e0 on white

Accessibility Guidelines:

  • Contrast ratio: At least 4.5:1 for normal text
  • Don't rely on color alone to convey meaning
  • Test with color blindness simulators
  • Use tools like WebAIM's Contrast Checker
/* Good: High contrast */
.good-contrast {
  background-color: #182848; /* Dark blue */
  color: #ffffff; /* White */
}

/* Bad: Low contrast */
.bad-contrast {
  background-color: #f0f0f0; /* Light gray */
  color: #ffffff; /* White */
}
